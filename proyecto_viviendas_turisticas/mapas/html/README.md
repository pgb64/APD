# Mapas HTML generados

En este directorio se generarán automáticamente los archivos HTML de los mapas (por ejemplo, mapas de calor) al ejecutar el script principal:

mapas/main.py


Estos archivos **NO están incluidos en el repositorio** porque se generan de forma automática y su tamaño es muy elevado (pueden superar el límite de archivos permitido por GitHub).

## Cómo generar los mapas

1. Asegúrate de tener todas las dependencias instaladas.
2. Ejecuta: mapas/main.py
3. El/los archivos HTML se guardarán en este directorio.

## Nota

Este directorio actúa únicamente como destino de salida para los mapas generados.  
Los archivos HTML resultantes deben considerarse **artefactos generados**, no código fuente.
