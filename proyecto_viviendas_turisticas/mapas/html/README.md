# Mapas HTML generados

En este directorio se generarán automáticamente el archivo HTML del mapa interactivo al ejecutar el script principal:

mapas/main.py

Este archivoa **NO está incluido en el repositorio** porque se genera de forma automática y su tamaño es muy elevado (pueden superar el límite de archivos permitido por GitHub).

## Cómo generar el mapa

1. Asegúrate de tener todas las dependencias instaladas.
2. Ejecuta: mapas/main.py
3. El archivo HTML se guardarán en este directorio.

## Nota

Este directorio actúa únicamente como destino de salida para los mapas generados.  
Los archivos HTML resultantes deben considerarse **artefactos generados**, no código fuente.
